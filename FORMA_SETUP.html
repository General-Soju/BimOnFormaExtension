🟣 BimOn AI — Autodesk Forma (Site Design) 설정 안내

Revit · Navisworks · AutoCAD는 설치만 하면 자동 연동됩니다. Forma Site Design만 클라우드 제품이라 아래 1회 등록이 필요합니다.

✓ 설치는 이미 끝났습니다. BimOn-Forma MCP가 Claude에 등록되었고, 확장 파일이 PC에 배포되었습니다. 이 페이지의 단계대로 Forma 계정에서 확장을 한 번만 등록하면 그 후로는 패널만 열면 동작합니다.

준비물 (2가지)

항목설명
Autodesk Forma 계정등록할 프로젝트가 있는 계정
APS 애플리케이션 1개없으면 aps.autodesk.com → Create App 으로 1개 생성

확장 URL (이 주소를 그대로 사용)

설치된 브리지가 확장을 직접 제공합니다. 별도 서버·호스팅이 필요 없습니다.

http://localhost:51737/standalone.html
⚠️ 반드시 http://localhost 로 입력하세요. Forma는 임베디드 뷰 URL로 https 또는 http://localhost만 허용합니다. http://127.0.0.1 이나 공개 https 주소는 거부됩니다.

등록 단계 (1회만)

  1. Forma에서 프로젝트 열기 → 좌측 확장 기능(Extensions) 아이콘 → + 확장 기능 추가
  2. App Store 우상단 ⚙ (톱니) → 확장 기능 생성(Create Extension)
  3. 이름 입력(예: BimOn AI) → 약관 동의 → Create
  4. ★ Owner를 "Myself only" → 본인 APS 애플리케이션으로 변경
    이래야 URL 입력 칸이 열립니다. (되돌릴 수 없음)
  5. Integration → Embedded views 에서:
    placementRIGHT_MENU_ANALYSIS_PANEL
    URLhttp://localhost:51737/standalone.html
  6. Who are allowed to use → Only specific projects → 사용할 프로젝트 ID(pro_xxxxx) 추가 → Save
  7. 프로젝트 App Store에서 BimOn AI 검색 → Add → 약관 Agree
  8. 우측 분석 패널의 확장 런처(큐브 아이콘) → BimOn AI 클릭
✓ 연결 성공 신호 — 패널에 🟢 Connected — ws://127.0.0.1:51737 + SDK loaded ✓ 가 뜨면 완료입니다. 이제 Claude에서 BimOn-Forma 도구가 동작합니다.

사용 중 유지 조건

  • 이 패널이 열려 있어야 Claude가 Forma를 제어합니다 (백그라운드 불가).
  • 브리지는 소켓이 1개라 패널은 한 곳에서만 열어주세요.
  • 쓰기(WRITE) 작업은 Forma 편집 권한이 있어야 동작합니다.

문제 해결

증상조치
패널이 비어 있음 / Missing ... originURL을 직접 연 것. 반드시 Forma 확장으로 등록해서 열어야 함
URL 입력란이 안 보임Owner가 "Myself only". APS 앱으로 변경(4단계)
Claude에서 not connected패널 미연결. 패널을 열고 🟢 Connected 확인
